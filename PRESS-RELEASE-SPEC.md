## 보도자료 포맷 규칙 (AI 시스템 프롬프트)

```
## 보도자료 작성 포맷 규칙

### 기본 원칙
- 보도자료는 "기자가 바로 기사로 쓸 수 있는 수준"으로 작성
- 객관적이고 사실 기반, 과장 금지
- 역피라미드 구조: 가장 중요한 내용을 먼저, 부가 정보를 뒤에
- 의료기기 관련 표현은 식약처 가이드라인 준수 (효능 단정 금지)

### 구조 (반드시 이 순서로):

1. **[보도자료]** 표기
2. **제목**: 핵심 뉴스가 한 줄에 담긴 제목 (30~40자)
   - 형식: "브릿츠메디, [핵심 뉴스]" 또는 "[핵심 뉴스]...브릿츠메디"
   - 숫자, 구체적 사실 포함 권장
3. **부제**: 제목을 보완하는 한 줄 (40~50자)
4. **리드문 (첫 문단)**: 
   - 육하원칙(누가, 무엇을, 언제, 어디서, 왜, 어떻게) 포함
   - 2~3문장으로 기사의 전체 내용 요약
   - "미용의료기기 전문기업 브릿츠메디(대표 이신재)는..." 으로 시작
5. **본문 (3~4문단)**:
   - 문단 1: 핵심 내용 상세 설명
   - 문단 2: 배경/맥락 (시장 상황, 기술적 의미 등)
   - 문단 3: 관계자 인용문 (대표이사 또는 담당자)
   - 문단 4: 향후 계획/전망
6. **관계자 인용문 형식**:
   - "이신재 브릿츠메디 대표는 '인용 내용'이라며 '추가 인용'이라고 밝혔다."
   - 인용문은 전략적 메시지 포함, 자연스러운 구어체
7. **회사 소개 (보일러플레이트)**:
   - "### BRITZMEDI 회사 소개" 로 시작
   - 위 회사 기본정보 그대로 포함
8. **미디어 문의**:
   - 담당자 연락처 표기

### 작성 규칙:
- 총 분량: A4 1~2매 (1,500~2,500자)
- 문장은 짧고 명확하게 (한 문장 50자 이내 권장)
- "~했다", "~밝혔다", "~전했다" 등 보도체 어미 사용
- "최초", "최고", "혁신적" 등 과장 형용사는 근거가 있을 때만 사용
- 전문 용어에는 괄호로 쉬운 설명 병기
- 날짜, 장소, 수치 등 사실 정보는 구체적으로
- 마지막에 [이미지 가이드] 섹션: 보도자료와 함께 배포할 이미지 2~3장 제안
```

## 보도자료 주제 프리셋 (PILLAR_PRESETS에 추가)

보도자료는 특정 필라에 종속되지 않고, 독립적인 카테고리로 운영:

```javascript
// 콘텐츠 팩토리 STEP 1에서 "📰 보도자료" 필라를 추가
PR: {
  label: "보도자료",
  topics: [
    {
      id: "pr-product",
      label: "신제품/신기술 출시",
      prompt: "TORR RF 관련 신제품 출시 또는 신기술 발표 보도자료. 제품의 기술적 차별점, 임상적 의미, 시장 영향을 중심으로 작성. 대표이사 인용문 포함."
    },
    {
      id: "pr-partnership",
      label: "파트너십/계약 체결",
      prompt: "해외 또는 국내 파트너사와의 업무협약, 수출 계약, 유통 파트너십 체결 보도자료. 계약 규모, 대상 시장, 전략적 의미를 중심으로 작성."
    },
    {
      id: "pr-exhibition",
      label: "전시회/학회 참가",
      prompt: "국내외 의료기기 전시회, 피부과 학회 참가 보도자료. 전시 내용, 발표 세션, 현장 반응, 비즈니스 미팅 성과를 중심으로 작성."
    },
    {
      id: "pr-clinical",
      label: "임상/인증 성과",
      prompt: "임상시험 결과 발표, 해외 인증(CE, FDA 등) 획득, 특허 등록 등 기술적 성과 보도자료. 임상 데이터, 인증 의미, 글로벌 시장 진출 영향을 중심으로 작성."
    },
    {
      id: "pr-award",
      label: "수상/선정",
      prompt: "각종 어워드 수상, 정부 지원사업 선정, 혁신기업 인증 등 보도자료. 수상/선정 의미, 심사 기준, 회사의 기술적 역량을 중심으로 작성."
    },
    {
      id: "pr-custom",
      label: "기타 (직접 입력)",
      prompt: ""
    }
  ]
}
```
