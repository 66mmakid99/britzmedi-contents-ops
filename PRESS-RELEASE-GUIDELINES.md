# BRITZMEDI 보도자료 작성 가이드라인

> AI가 보도자료를 생성할 때 반드시 참조하는 규칙서.
> 이 문서의 모든 규칙은 생성 프롬프트와 검수 프롬프트 양쪽에 적용된다.

---

## 1. 글쓰기 기본 — 국립국어원 어문 규정 준수

모든 콘텐츠는 국립국어원 어문 규정을 기본으로 따른다. 다른 모든 규칙보다 우선 적용.

### 1-1. 맞춤법

- 띄어쓰기 정확히 (예: "할 수 있다" ○ / "할수있다" ✕)
- 사이시옷 규정 준수 (예: "횟수" ○ / "회수" ✕)
- "되" vs "돼" 구분 (예: "진행됐다" ○ / "진행됬다" ✕)
- "로서" vs "로써" 구분 (자격 vs 수단)
- 된소리 표기 주의

### 1-2. 외래어 표기법

- 국립국어원 외래어 표기법 기준
- 예: "솔루션" ○ / "솔류션" ✕
- 예: "에스테틱" ○ / "에스테틱스" ✕ (업계 통용 표기 허용)
- 의료 전문용어는 대한의학회 의학용어집 기준 우선

### 1-3. 문장 부호

- 쌍점(:) 뒤 띄어쓰기
- 쉼표(,) 뒤 띄어쓰기
- 괄호 안팎 띄어쓰기 규정 준수
- 큰따옴표 " " (직접 인용), 작은따옴표 ' ' (강조, 인용 내 인용)

### 1-4. 피해야 할 흔한 오류

- "~에 대해" 남용 금지 → 구체적 서술어 사용
- "~것으로 보인다" 남용 금지 → 보도자료에서는 단정적 서술
- 같은 조사/어미 연속 반복 금지 (예: "~하며, ~하며, ~하며" ✕)

## 2. 보도문체 (Press Release Style) 엄격 준수

보도자료는 반드시 보도문체로 작성한다.

### 2-1. 문장 종결

| 허용 | 예시 |
|------|------|
| 기본 종결 | "~했다", "~밝혔다", "~전했다", "~설명했다" |
| 계획 서술 | "~할 예정이다", "~할 계획이다" |
| 기대 표현 | "~될 것으로 기대된다" (본문 내 1회 이내) |

| 절대 금지 | 예시 |
|-----------|------|
| 경어체 | "~합니다", "~입니다", "~했습니다" |
| 구어체 | "~하고 있어요", "~인데요", "~거든요" |
| 명사형 종결 | "~했음", "~임" |

### 2-2. 주어 표기

- 첫 문장: "브릿츠메디(대표 이신재)는" — 회사명 + 대표명
- 이후: "브릿츠메디는" 또는 "동사(同社)는"
- 3인칭 시점 유지 — "우리", "저희" 절대 금지

### 2-3. 인용문 형식

```
이신재 브릿츠메디 대표는 '인용문 내용'이라며 '인용문 내용'이라고 밝혔다.
```

또는:
```
이신재 대표는 '인용문 내용'이라고 전했다.
```

- 인용문은 작은따옴표 ' ' 사용 (보도 관행)
- "~라며 ~라고 밝혔다" 또는 "~라고 전했다"
- 인용문 안에서도 의료법 금지어 사용 불가

### 2-4. 문단 구조

- 한 문단 2~4문장
- 한 문장 40~60자 이내 (지나치게 긴 문장 금지)
- 역피라미드 구조: 핵심 → 상세 → 배경 → 향후 계획

### 2-5. 수동태/피동 자제

| 수동태 (✕) | 능동태 (○) |
|-----------|-----------|
| "체결되었다" | "체결했다" |
| "진행될 예정이다" | "진행할 예정이다" |
| "확보된 것으로 평가된다" | "확보했다" |

### 2-6. 출처 불명 표현 금지

| 금지 표현 | 이유 | 대안 |
|----------|------|------|
| "~로 평가된다" | 누구의 평가? | 삭제하거나 구체적 출처 명시 |
| "~로 알려져 있다" | 누구에게? | 삭제하거나 구체적 출처 명시 |
| "업계에서는 ~로 보고 있다" | 근거 없음 | 삭제 |
| "~로 주목받고 있다" | 누구의 주목? | 삭제하거나 구체적 근거 명시 |
| "~로 기대된다" | 남용 주의 | 본문 내 최대 1회만 허용 |

## 3. 외래어/영문 표기 규칙 (절대 규칙)

### 3-1. 원칙

**모든 영문 표현은 '국문(영문)' 순서로, 본문에서 첫 등장 시 1회만 병기한다. 두 번째부터는 국문으로만 표기한다.**

### 3-2. 적용 목록

| 첫 등장 (1회만) | 이후 전부 |
|----------------|----------|
| 미국 식품의약국(FDA) | 미국 식품의약국 |
| 에너지 기반 디바이스(EBD) | 에너지 기반 디바이스 |
| 토로이달(TOROIDAL) | 토로이달 |
| 토르RF(TORR RF) | 토르RF |
| 루미노웨이브(LuminoWave) | 루미노웨이브 |
| 브릿츠메디(BRITZMEDI) | 브릿츠메디 |

※ 제품명에 영문이 포함된 경우(토르RF)는 고유명사이므로 그대로 유지.

### 3-3. 금지 사항

| 금지 | 올바른 표기 |
|------|-----------|
| "FDA 승인" (영문 단독, 첫 등장) | "미국 식품의약국(FDA) 승인" |
| "TOROIDAL 고주파" (영문이 먼저) | "토로이달(TOROIDAL) 고주파" |
| "EBD 제품" (영문 단독, 첫 등장) | "에너지 기반 디바이스(EBD) 제품" |
| 2회 이상 영문 병기 | 첫 등장에서 1회만 병기, 이후 국문만 |

### 3-4. 예시 — 올바른 본문

```
메디컬 에스테틱 디바이스 전문기업 브릿츠메디(BRITZMEDI, 대표 이신재)는
미국 식품의약국(FDA) 승인을 받은 토로이달(TOROIDAL) 고주파 기술 기반
에너지 기반 디바이스(EBD) 토르RF(TORR RF)를 전시한다고 밝혔다.

브릿츠메디는 이번 학회에서 토르RF를 중심으로 부스를 구성했으며,
신제품 루미노웨이브(LuminoWave)도 함께 선보였다. 토로이달 고주파 기술은
기존 RF 기술 대비 균일한 에너지 분포 특성을 갖추고 있다.
```

↑ 두 번째 문단: 전부 국문만 사용. 영문 병기 없음. 루미노웨이브만 첫 등장이라 병기.

## 4. 의료법 준수

### 4-1. 절대 금지 표현

의료기기 광고에서 절대 사용 금지하는 표현:

| 금지어 | 대체 표현 |
|--------|----------|
| 극대화 | 개선, 향상, 촉진 |
| 최소화 | 감소, 줄이기 |
| 최고 | 높은 수준의, 우수한 |
| 최초 (근거 없이) | (구체적 근거 필요) |
| 완벽 | 우수한, 정밀한 |
| 획기적 | 차별화된, 발전된 |
| 혁명적 | 혁신적 (주의하여 사용) |
| 완치 | 개선, 호전 |
| 100% | 높은 비율의 |
| 확실한 효과 | 기대되는 효과 |
| 부작용 없음 | (사용 금지 — 모든 의료기기에 부작용 가능) |
| 안전 보장 | 안전성 확인, 안전성을 갖춘 |

### 4-2. 효능 단정 금지

| 금지 (✕) | 허용 (○) |
|----------|---------|
| "~를 치료합니다" | "~개선에 기여한다" |
| "~가 사라집니다" | "~감소에 도움이 된다" |
| "부작용이 없다" | 사용 금지 |
| "확실한 효과" | "기대되는 효과" |

### 4-3. 비교 광고 금지

- 타사 제품명 직접 언급 금지
- "기존 제품 대비" 허용 (자사 제품 간 비교)
- "타사 대비" 금지

## 5. 보도자료 구조 규칙

### 5-1. 역피라미드 구조

```
[제목] 핵심 한 줄 — 회사명 + 무엇을 했는지
[부제목] 보충 설명 — 구체적 수치나 의미
[리드문] 육하원칙 — 누가, 무엇을, 언제, 어디서 (1문단, 2~3문장)
[본문] 상세 — 제품 설명, 계약 내용, 기술 설명 (2~3문단)
[인용문] 대표 코멘트 — 의미와 비전 (1문단)
[향후 계획] 다음 스텝 (1문단, 소스에 있을 때만)
[보일러플레이트] 회사 소개 (항상 동일)
[연락처] 미디어 문의 (항상 동일)
[사진 가이드] 첨부 이미지 제안 2~3장
[첨부파일 가이드] 첨부할 문서 목록
[태그] 키워드 5~7개
```

### 5-2. 필수 포함 항목

| 항목 | 필수 | 비고 |
|------|------|------|
| 제목 | ● | |
| 부제목 | ● | |
| 리드문 (육하원칙) | ● | |
| 본문 | ● | 2~4문단 |
| 대표 인용문 | ● | 소스에 없으면 AI가 3개 제안 |
| 보일러플레이트 | ● | 자동 삽입 |
| 연락처 | ● | 자동 삽입 |
| 사진 가이드 | ○ | |
| 첨부파일 가이드 | ○ | |
| 태그 | ○ | |

### 5-3. 제목/리드문/본문 일관성

- 제목에 언급한 제품은 본문에서 반드시 1줄 이상 설명
- 리드문에 언급한 사실은 본문에서 상세 전개
- 본문에서 갑자기 리드문에 없는 새 주제 등장 금지

## 6. 시제 규칙

### 6-1. 사전 배포 (예고형)

- 전체 미래 시제: "~할 예정이다", "~를 선보인다", "~를 개최한다"
- 과거 시제 금지 (아직 안 일어난 일이므로)
- 예외: 과거 실적/인증은 과거 시제 허용 ("FDA 승인을 받은")

### 6-2. 사후 배포 (리뷰형)

- 전체 과거 시제: "~했다", "~밝혔다", "~를 전시했다"
- 향후 계획 문장만 미래 시제 허용 ("~할 예정이다")
- 미래 시제와 과거 시제 혼재 금지 (향후 계획 문단 제외)

## 7. 팩트 사용 규칙 (할루시네이션 방지)

### 7-1. 정보 소스 3단계

| 우선순위 | 소스 | 사용 범위 |
|---------|------|----------|
| 1순위 | 사용자 입력 소스 (확인된 팩트) | 뼈대 — 핵심 사실, 수치, 일정 |
| 2순위 | 지식베이스 등록 정보 | 살 — 제품 설명, 기술 설명, 회사 배경 |
| 3순위 | 없음 | 절대 만들지 않음 → [입력 필요] 플레이스홀더 |

### 7-2. 절대 금지 (날조)

- 소스에 없는 숫자 (참석자 수, 미팅 건수, 매출액 등)
- 소스에 없는 인용문
- 소스에 없는 현장 반응, 참가자 반응
- 소스에 없는 계약 상대방 회사 상세 소개
- 소스에 없는 시장 확장 계획 (예: "중동 진출", "주변국 확장")
- 소스에 없는 평가 ("~로 평가된다", "~로 주목받고 있다")

### 7-3. 허용 (지식베이스 기반 살 붙이기)

- 자사 제품 기술 설명 (지식베이스에 등록된 것)
- 회사 연혁, 설립 연도, 대표 정보
- 기존 인증 사실 (FDA 승인 등 — 지식베이스에 등록된 것)
- 기술의 일반적 특성 설명 (지식베이스에 등록된 것)

### 7-4. 분량 규칙

- 적정 분량: A4 1~1.5매 (1,000~2,000자)
- 팩트가 부족해서 분량이 안 채워지면 억지로 늘리지 말고 짧게 끝낸다
- 짧은 보도자료 > 날조가 포함된 긴 보도자료

## 8. 검수 체크리스트

생성된 보도자료를 아래 항목으로 검수한다. AI가 자동 수정할 수 있는 건 자동 수정하고, 사용자 입력이 필요한 건 ⚠️로 표시한다.

### 자동 수정 항목 (✅)

| 카테고리 | 체크 항목 | 수정 방법 |
|---------|----------|----------|
| [WRITING] | 맞춤법/띄어쓰기 오류 | 국립국어원 기준 자동 수정 |
| [STYLE] | 경어체/구어체 사용 | 보도문체로 변환 |
| [STYLE] | 출처 불명 표현 | 삭제 또는 사실 기반 표현으로 변환 |
| [STYLE] | 수동태 과다 | 능동태로 변환 |
| [NOTATION] | 영문 단독 사용 (첫 등장) | 국문(영문) 형태로 변환 |
| [NOTATION] | 영문이 국문보다 앞에 옴 | 순서 변경 |
| [NOTATION] | 2회 이상 영문 병기 | 2회차부터 국문으로만 변환 |
| [LAW] | 의료법 금지어 사용 | 허용 표현으로 대체 |
| [LAW] | 효능 단정 표현 | 완화 표현으로 대체 |
| [TIME] | 시제 불일치 | 시점에 맞는 시제로 통일 |
| [STRUCT] | 보일러플레이트 누락 | 자동 삽입 |
| [STRUCT] | 연락처 누락 | 자동 삽입 |
| [FACT] | 소스에 없는 내용 발견 | 해당 문장 삭제 |

### 사용자 입력 필요 항목 (⚠️)

| 카테고리 | 체크 항목 | 표시 방법 |
|---------|----------|----------|
| [FACT] | 지식베이스에 미등록 제품 정보 | [입력 필요: 제품명 상세] |
| [FACT] | 행사 정식 명칭 미확인 | ⚠️ 사실 확인 필요 |
| [FACT] | 날짜/장소 미입력 | [입력 필요: 항목명] |
| [QUOTE] | 대표 인용문 미입력 | 3개 제안 → 선택 |

## 9. 보도자료 보일러플레이트 (항상 동일)

```
브릿츠메디(BRITZMEDI)는 2017년 설립된 메디컬 에스테틱 디바이스 전문기업으로,
```
