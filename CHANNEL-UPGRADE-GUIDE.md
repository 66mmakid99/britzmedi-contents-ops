# BRITZMEDI Content Factory — 채널별 콘텐츠 품질 개선 가이드

> **목적:** 이 문서는 Claude Code가 `src/constants/prompts.js`의 `getRepurposePrompt` 함수 내  
> 각 채널 프롬프트를 교체하기 위한 **분석 + 실행 지침**입니다.  
> **적용 파일:** `src/constants/prompts.js` → `getRepurposePrompt(channel, ...)`  
> **현재 문제:** 모든 채널이 "보도자료를 약간 다듬은" 수준으로, 채널 고유의 톤·구조·목적이 반영되지 않음

---

## 공통 진단: "이도저도 아닌" 이유

| 문제 | 설명 |
|------|------|
| **보도자료 복사·붙여넣기 냄새** | 5개 채널 모두 원본 보도자료의 문장 구조/순서를 거의 그대로 따름. "재가공"이 아니라 "축약" 수준 |
| **독자(Reader) 부재** | 누구에게 말하는지 불분명. 이메일은 원장님, 블로그는 일반 대중, 인스타는 MZ세대, 링크드인은 업계 전문가 — 이 구분이 없음 |
| **채널 네이티브 문법 무시** | 네이버 블로그의 "~요"체, 인스타의 짧은 훅, 카카오톡의 이모지, 링크드인의 인사이트 톤 등 플랫폼 고유 문법 미적용 |
| **CTA(행동 유도) 부재** | "자세히 보기 → britzmedi.co.kr" 정도가 전부. 채널별 목적에 맞는 CTA 설계 없음 |

## 1. 이메일 뉴스레터

### 1-1. 현재 상태 분석

```
현재 글자수: 1,270자 (권장 1,500~2,500자) → 부족
```

**문제점 상세:**

- **보도자료 재탕 느낌:** "지난 4월 15일, 브릿츠메디는 태국 방콕의 더마 솔루션스(Derma Solutions)사와 독점유통 계약을 체결했습니다" → 보도자료 1단락과 거의 동일
- **"원장님" 타깃인데 원장님 관점 부재:** 원장님이 이 뉴스를 왜 읽어야 하는지, 원장님 클리닉에 어떤 의미인지 연결 없음
- **소제목 톤이 딱딱:** "태국 시장 진출, 왜 의미 있는가" → 보도자료 소제목 그대로. 뉴스레터는 대화체 소제목이 적합
- **마무리가 갑작스러움:** 핵심 포인트 나열 후 연락처로 바로 끝남. 다음 액션(CTA) 없음
- **시각적 구분 요소 부재:** 뉴스레터는 하이라이트 박스, 핵심 숫자 강조, 구분선 등이 필요

### 1-2. 이메일 뉴스레터 — 채널 가이드라인

**채널 목적:** 피부과/에스테틱 원장님에게 브릿츠메디의 최신 소식을 전하되, "우리 클리닉에 어떤 의미가 있는가" 관점으로 가치를 전달

**독자 페르소나:**
- 피부과/에스테틱 원장님 (의사)
- 바쁘고, 새로운 장비/기술 트렌드에 관심
- 직접적 이익(매출, 환자 만족)에 민감
- 업계 동향을 빠르게 파악하고 싶어함

**톤앤매너:**
- 존칭 사용 (~습니다, ~하셨을 겁니다)
- 전문적이되 따뜻한 톤. 딱딱한 보고서 ✕, 친근한 전문가 브리핑 ○
- "원장님"이라는 호칭을 자연스럽게 2~3회 사용
- 업계 전문용어 적절히 활용 (EBD, 토로이달 고주파 등)

**필수 구조 (6파트):**

```
① 인사 + 한 줄 훅 (2~3문장)
   "원장님, 안녕하세요. 브릿츠메디입니다."
   + 이번 소식이 원장님에게 왜 의미 있는지 한 줄 요약
   예: "동남아시아 메디컬 에스테틱 시장에서 토르RF가 인정받기 시작했습니다."

② 핵심 뉴스 요약 (3~4문장)
   누가, 무엇을, 왜 — 팩트 중심 간결 요약
   반드시 기간+수량 쌍 포함 (예: "3년간 연 300대")

③ 원장님 관점 인사이트 (4~6문장) ★ 가장 중요
   "이 소식이 국내 피부과 원장님들께 의미 있는 이유"
   - 해외 시장 검증 = 국내 장비 신뢰도 향상
   - 글로벌 임상 현장에서의 기술력 인정
   - 원장님 클리닉의 장비 선택에 참고할 포인트
   ※ 보도자료에 없는 원장님 관점의 해석을 추가해야 함

④ 기술/제품 하이라이트 (3~4문장)
   토르RF의 핵심 기술 차별점을 원장님 입장에서 설명
   - 시술 효과, 다운타임, 환자 만족도 관점
   - 전문 용어 사용 가능하되 설명적으로

⑤ 핵심 포인트 정리 (불릿 3개)
   · 팩트1
   · 팩트2  
   · 팩트3

⑥ 마무리 + CTA (2~3문장)
   "토르RF에 대해 더 자세히 알고 싶으시다면 담당자에게 문의해주세요."
   담당자: 이성호 CMO / sh.lee@britzmedi.co.kr / 010-6525-9442
```

**금지사항:**
- 보도자료 문장을 그대로 복사하지 않을 것
- "빠르게 성장하는" 같은 클리셰 반복 금지
- 소제목에 대괄호([ ]) 사용 금지
- "뷰티 디바이스", "K-뷰티", "미용기기" 금지

## 2. 네이버 블로그

### 2-1. 현재 상태 분석

```
현재 글자수: 1,880자 (권장 2,000~3,500자) → 약간 부족
```

**문제점 상세:**

- **"~다" 체 사용:** 네이버 블로그는 "~요" 체(해요체)가 표준. 현재는 보도자료의 "~다" 체 그대로
- **소제목이 딱딱:** "[태국 피부과 시장 진출, 3년간 연 300대 규모 계약]" → 대괄호 + 나열식. 블로그는 호기심 유발형 소제목 필요
- **독자 관점 부재:** "브릿츠메디는 ~했다" 기업 관점 일색. 블로그는 독자가 "왜 이 글을 읽어야 하는가" 느낌이어야 함
- **이미지 플레이스홀더:** [IMAGE: 설명] 형식은 좋으나, 실제 네이버 블로그 레이아웃처럼 자연스러운 배치 안내 부족
- **SEO 키워드 활용:** 하단에 키워드 나열은 있으나, 본문 내 자연스러운 키워드 삽입 전략 없음
- **인트로/아웃트로 부재:** 블로그는 도입(왜 이 주제인지)과 마무리(독자에게 질문/액션)가 핵심

### 2-2. 네이버 블로그 — 채널 가이드라인

**채널 목적:** 네이버 검색을 통해 유입되는 의료기기/에스테틱 관심층에게 브릿츠메디의 뉴스를 "읽을거리"로 전달. SEO 최적화 + 정보성 콘텐츠

**독자 페르소나:**
- 의료기기 업계 관계자, 피부과 관계자, 에스테틱 관심 일반인
- 네이버에서 "토르RF", "메디컬에스테틱디바이스", "의료기기수출" 등 검색해서 유입
- 긴 글도 읽지만, 스캔이 가능해야 함 (소제목 + 이미지로 훑어보기)

**톤앤매너:**
- **해요체 필수** (~합니다/~해요/~인데요/~거든요)
- 정보성 + 약간의 친근함. "전문 매거진 기사" 느낌
- 1인칭 사용 가능: "브릿츠메디가 최근 발표한 소식을 전해드릴게요"
- 독자에게 말 걸기: "혹시 ~에 대해 들어보셨나요?" 같은 질문형 활용

**필수 구조 (7파트):**

```
① 제목 (SEO 최적화)
   핵심 키워드 + 구체적 수치 포함
   예: "브릿츠메디 토르RF, 태국 독점유통 계약 체결 - 3년간 연 300대 공급"
   ※ 40자 내외, 네이버 검색 노출 최적화

② 부제목
   제목을 보완하는 한 줄 요약
   예: "브릿츠메디 토르RF, 태국 독점유통 계약 - 3년간 연 300대 공급"

③ 도입부 (3~4문장) — 독자 관심 유도
   현재 트렌드나 시장 상황으로 시작 → 이 소식이 왜 주목할 만한지 연결
   예: "동남아시아 메디컬 에스테틱 시장이 빠르게 성장하고 있다는 건 업계에서 이미 잘 알려진 이야기인데요. 그 중심에서 국내 의료기기 기업의 존재감이 커지고 있어요."
   
④ 본문 섹션 1: 뉴스 핵심 (4~6문장)
   소제목: 호기심 유발형 (질문형 또는 감탄형)
   예: "태국 피부과 시장, 왜 지금 주목받을까요?"
   팩트 중심이되 해요체로 자연스럽게
   
   [IMAGE: 계약 체결식 현장 사진 또는 관련 이미지]

⑤ 본문 섹션 2: 기술/제품 심화 (4~6문장)
   소제목 예: "FDA 승인 토로이달 고주파, 어떤 점이 다를까요?"
   기술적 차별점을 이해하기 쉽게 설명
   전문용어는 쓰되 괄호로 부연
   
   [IMAGE: 토르RF 제품 이미지]

⑥ 본문 섹션 3: 의미/전망 (3~4문장)
   소제목 예: "국내 의료기기 업계에 주는 시사점"
   이 뉴스의 업계 차원 의미, 향후 전망

⑦ 마무리 (2~3문장) + 핵심 요약 불릿
   독자에게 한마디: "앞으로의 행보도 주목해 볼 만하겠죠?"
   · 핵심 포인트 3개
   
   브릿츠메디 공식 홈페이지: www.britzmedi.co.kr

   SEO: 키워드1, 키워드2, ... (쉼표 구분, 7~12개)
```

**SEO 키워드 규칙:**
- 본문 내 핵심 키워드 자연스럽게 3~5회 반복
- 하단 SEO 태그: 브랜드명 + 제품명 + 업종 + 기술명 + 지역명 조합
- 제목에 반드시 핵심 키워드 1개 이상 포함

**금지사항:**
- "~다" 체(경어체/합쇼체) 사용 금지 → 반드시 해요체
- 대괄호([ ]) 소제목 금지 → 자연어 소제목 사용
- 보도자료 문장 그대로 복사 금지
- "뷰티 디바이스", "K-뷰티", "미용기기" 금지

## 3. 카카오톡 채널

### 3-1. 현재 상태 분석

```
현재 글자수: 345자 (권장 300~500자) → 적정
```

**문제점 상세:**

- **톤이 너무 건조:** "브릿츠메디가 태국 더마 솔루션스와 토르RF 독점유통 계약을 체결했습니다" → 보도자료 1문장 축약판
- **이모지 제로:** 카카오톡 채널 메시지에서 이모지는 필수 요소. 가독성과 친근감 모두에 기여
- **줄바꿈/여백 부족:** 모바일 화면에서 한 덩어리 텍스트는 읽기 어려움
- **CTA가 너무 약함:** "자세히 보기 → britzmedi.co.kr"는 클릭 유도력 낮음
- **독자 편익 부재:** "계약 체결" 자체는 기업 소식. 구독자가 왜 관심 가져야 하는지 없음

### 3-2. 카카오톡 채널 — 채널 가이드라인

**채널 목적:** 카카오톡 채널 구독자에게 핵심 소식을 빠르고 친근하게 전달. 짧지만 임팩트 있게, 링크 클릭 유도

**독자 페르소나:**
- 브릿츠메디에 관심 있는 업계 관계자, 거래처, 잠재 고객
- 모바일에서 카카오톡 알림으로 수신
- 3~5초 안에 내용 파악 가능해야 함
- 관심이 생기면 링크 클릭

**톤앤매너:**
- **해요체** (~해요/~했어요/~드려요)
- 친근하고 가볍되 전문성은 유지
- 이모지 적극 활용 (문단 시작마다 1개씩, 총 3~5개)
- 짧은 문장, 넉넉한 줄바꿈

**필수 구조 (4파트):**

```
① 한 줄 헤드라인 + 이모지 (1문장)
   🎉 브릿츠메디, 태국 독점유통 계약 체결!

② 핵심 내용 (2~3문장, 문장마다 줄바꿈)
   🇹🇭 태국 더마 솔루션스와 토르RF 독점유통 계약을 맺었어요.
   3년간 연 300대 규모로, 올 하반기부터 태국 피부과에 납품을 시작해요.
   
③ 의미/한줄 인사이트 (1~2문장)
   🌏 동남아시아 메디컬 에스테틱 시장 공략의 첫 발걸음이에요.

④ CTA (1문장 + 링크)
   👉 자세한 소식 확인하기
   britzmedi.co.kr
```

**이모지 사용 규칙:**
- 헤드라인: 🎉, 🔔, 📢, ✨ 중 상황에 맞게 1개
- 내용: 주제 관련 이모지 (국기, 🏥, 💉, 📋 등)
- 전망/의미: 🌏, 🚀, 💡, 📈 등
- CTA: 👉 고정

**금지사항:**
- 3문장 이상 연속 텍스트 덩어리 금지 (반드시 줄바꿈)
- "~했다", "~이다" 같은 격식체 금지
- 이모지 없는 메시지 금지 (최소 3개)
- "뷰티 디바이스", "K-뷰티", "미용기기" 금지

## 4. 인스타그램

### 4-1. 현재 상태 분석

```
현재 글자수: 136자 (권장 50~150자) → 적정
```

**문제점 상세:**

- **훅(Hook) 부재:** 첫 줄이 "브릿츠메디가 태국 더마 솔루션스와 토르RF 독점유통 계약을 체결했습니다." → 피드에서 스크롤 멈출 이유 없음
- **이모지 제로:** 인스타그램에서 이모지 없는 캡션은 기업 공지문 느낌
- **해시태그 과다 + 비효율:** 16개 해시태그 중 #에너지기반디바이스, #EBD 같은 것은 검색량 거의 없음
- **이미지 가이드만 있고 캡션 전략 부재:** 인스타 캡션은 "첫 줄에서 승부"가 핵심
- **톤이 격식체:** "본격화합니다" → 인스타에서는 어울리지 않음

### 4-2. 인스타그램 — 채널 가이드라인

**채널 목적:** 브릿츠메디의 브랜드 이미지 구축 + 업계 존재감. 비주얼 중심, 캡션은 임팩트 있게 짧게

**독자 페르소나:**
- 메디컬 에스테틱 업계 관계자, 의료기기 관심층
- 피드를 빠르게 스크롤하며, 첫 1~2줄에서 멈출지 결정
- 해시태그로 콘텐츠 탐색
- 이미지/비주얼이 핵심, 캡션은 보조

**톤앤매너:**
- **간결하고 임팩트 있는 문체** — 짧은 문장, 강한 첫 줄
- 이모지로 시각적 리듬 (각 줄 시작 또는 핵심 포인트에)
- "~합니다" 보다 "~!" 또는 "~요" 체
- 프로페셔널하되 약간의 자신감/에너지

**필수 구조 (4파트):**

```
① 훅 라인 (1문장) ★ 가장 중요
   피드에서 "더 보기" 누르게 만드는 첫 줄
   숫자, 성과, 질문 중 하나로 시작
   예: "🇹🇭 태국 진출, 3년간 연 300대 독점 공급 계약 체결!"
   예: "동남아 메디컬 에스테틱 시장, 토르RF가 갑니다 🚀"

② 핵심 팩트 (2~3줄, 줄바꿈)
   ✅ 태국 더마 솔루션스와 독점유통 계약
   ✅ 3년간 연 300대 규모
   ✅ 올 하반기 태국 피부과 납품 시작

③ 이미지 가이드 (1문장)
   🎨 이미지: [이미지 설명 + 스타일 가이드]

④ 해시태그 (2줄)
   줄바꿈 후 해시태그 배치
   브랜드 태그(3~4개) + 업종 태그(4~5개) + 트렌드 태그(2~3개)
```

**해시태그 전략:**
- **브랜드 (고정):** #브릿츠메디 #BRITZMEDI #토르RF #TORRRF
- **업종 (뉴스별 선택 4~5개):** #메디컬에스테틱 #의료기기 #피부과 #에스테틱장비 #의료기기수출
- **트렌드/맥락 (뉴스별 선택 2~3개):** #태국진출 #동남아시장 #글로벌진출 #FDA승인
- **총 10~12개** (15개 이상은 스팸 느낌)
- **검색량 없는 태그 제거:** #EBD, #에너지기반디바이스, #토로이달고주파, #콜라겐리모델링 같은 것은 제거

**금지사항:**
- 3줄 이상 연속 텍스트 금지
- 첫 줄이 "브릿츠메디가 ~했습니다" 같은 평서문 시작 금지 (훅이 아님)
- 해시태그 15개 초과 금지
- 검색량 없는 해시태그 금지 (#에너지기반디바이스, #EBD, #토로이달고주파 등)
- "뷰티 디바이스", "K-뷰티", "미용기기" 금지

> (스크린샷 미제공이나, 기존 프롬프트 패턴으로 미루어 동일한 문제 예상)

### 5-2. 링크드인 — 채널 가이드라인

**채널 목적:** B2B 글로벌 네트워크에서 브릿츠메디의 전문성과 비즈니스 성과를 공유. 업계 전문가/잠재 파트너/투자자 대상

**독자 페르소나:**
- 의료기기 업계 전문가, 해외 디스트리뷰터, 투자자, 비즈니스 파트너
- 업계 인사이트와 비즈니스 스토리에 관심
- 글로벌 오디언스 포함 → 한영 혼용 자연스러움
- "이 회사/사람에게서 배울 게 있나?" 관점으로 읽음

**톤앤매너:**
- **프로페셔널 + 스토리텔링** — 격식 있되 인간적
- 1인칭 또는 회사 관점의 서사 ("We're excited to..." 또는 "브릿츠메디는 오늘...")
- 비즈니스 인사이트 제공: "왜 이 결정을 했는가", "이 시장의 기회는 무엇인가"
- 영어 키워드/문구 자연스럽게 혼용 (예: "go-to-market strategy", "market entry")
- 겸손하되 자신감 있는 톤

**필수 구조 (5파트):**

```
① 훅 (1~2문장)
   성과 + 의미를 한 문장으로
   예: "브릿츠메디가 태국 시장에 본격 진출합니다."
   또는 영문 혼용: "BRITZMEDI x Derma Solutions — 3-year exclusive distribution in Thailand 🇹🇭"

② 스토리/맥락 (3~4문장)
   왜 이 파트너십인지, 왜 이 시장인지
   비즈니스 의사결정의 배경과 전략적 의미
   데이터/수치 포함 (3년, 연 300대, 하반기 납품 시작)

③ 인사이트/학습 포인트 (2~3문장) ★ 링크드인 핵심
   이 경험에서 업계가 배울 수 있는 점
   예: "동남아시아 메디컬 에스테틱 시장은 연 15% 이상 성장 중.
   현지 유통 파트너와의 협업이 시장 진입의 핵심이라는 걸 다시 확인했습니다."
   
④ 전망/포부 (1~2문장)
   앞으로의 계획, 비전
   예: "이번 태국 진출을 시작으로 동남아 전역으로 확대해 나가겠습니다."

⑤ 해시태그 (1줄)
   #BRITZMEDI #TORRRF #MedicalAesthetics #MedicalDevice #Thailand
   #GlobalExpansion #EBD #Dermatology
```

**한영 혼용 규칙:**
- 제목/훅: 영문 포함 가능
- 본문: 한국어 기반 + 핵심 비즈니스 용어 영문 (exclusive distribution, market entry, EBD 등)
- 해시태그: **영문 위주** (링크드인은 글로벌 플랫폼)

**해시태그 전략:**
- **영문 위주 8~10개**
- 브랜드: #BRITZMEDI #TORRRF
- 업종: #MedicalAesthetics #MedicalDevice #Dermatology #EBD
- 맥락: #Thailand #GlobalExpansion #Partnership #B2B
- 한글 태그는 1~2개만: #의료기기 #메디컬에스테틱

**금지사항:**
- 단순 뉴스 나열 (보도자료 복사) 금지
- 인사이트/학습 포인트 없는 포스트 금지
- 한글 해시태그만 사용 금지 (영문 필수)
- 이모지 과다 사용 금지 (0~2개 적정, 카카오톡/인스타와 차별)
- "뷰티 디바이스", "K-뷰티", "미용기기" 금지
