# BRITZMEDI 채널 콘텐츠 작성 지침 v1

> **이 문서는 보도자료를 5개 채널로 재가공할 때의 절대 지침이다.**
> 채널별 생성, 검수, 자동수정의 모든 단계에서 이 문서를 준수한다.
> 보도자료 지침(PRESS-RELEASE-GUIDELINES)과 함께 적용된다.
>
> 최종 수정: 2026-02-16

## 0. 공통 원칙 — 모든 채널 공통

### 0-1. 소스 활용 원칙

모든 채널 콘텐츠는 **보도자료 본문**을 원본으로 재가공한다.
보도자료에 없는 내용을 새로 만들지 않는다.

```
✅ 사용 가능한 소스:
1순위: 보도자료 본문 (확정된 팩트)
2순위: 지식베이스에 등록된 정보 (제품 정보, 회사 정보)
3순위: 없으면 → 쓰지 않는다

❌ 절대 금지:
- 설문조사 데이터를 맥락 없이 끌어오기
- 소스에 없는 시장 분석, 통계, 전망 날조
- 보도자료에 없는 구체적 숫자 추가
```

### 0-2. 설문조사 데이터 사용 규칙

지식베이스의 113명 설문조사 결과는 **특정 조건에서만** 사용한다:

```
✅ 사용 가능한 경우:
- 네이버 블로그: 소제목 하나에서 "소비자 인사이트"로 자연스럽게 연결할 때
- LinkedIn: 전문가 대상 산업 인사이트에서 데이터 근거로 활용할 때
- 단, 반드시 "브릿츠메디가 실시한 113명 대상 소비자 설문에 따르면" 출처 명시

❌ 사용 금지:
- 카카오톡 카드뉴스 (공간 부족, 맥락 없이 숫자만 나열됨)
- 인스타그램 (캐러셀 공간 부족, 시각 중심 채널)
- 이메일 뉴스레터 인트로 (첫인상에 설문 데이터 부적절)
- 보도자료와 직접 관련 없는 맥락에서 억지로 연결
```

### 0-3. 의료법 준수 (전 채널 동일)

보도자료 지침의 의료법 금지어 규칙을 그대로 적용한다.
채널이 달라도 의료기기 관련 내용이므로 동일 기준.

### 0-4. 영문 표기 (전 채널 동일)

첫 등장 시 `국문(영문)`, 이후 국문만.
단, 인스타그램 해시태그와 LinkedIn 영문 키워드는 예외.

### 0-5. 팩트 비율 검수 기준

```
팩트 비율 = (소스/지식베이스에 근거한 문장 수) / (전체 문장 수)

보도자료: 100% 목표 (팩트 기반 매체)
네이버 블로그: 80% 이상 (해설/맥락 추가 허용)
LinkedIn: 70% 이상 (산업 인사이트, 전문가 해석 허용)
이메일 뉴스레터: 80% 이상
카카오톡: 90% 이상 (짧으니까 팩트 위주)
인스타그램: 85% 이상 (짧으니까 팩트 위주)
```

## 1. 카카오톡 카드뉴스

### 1-1. 채널 성격

```
목적: 핵심 뉴스를 빠르게 전달 (스크롤 1-2초 내 파악)
독자: 내부 공유용, 파트너사 전달용, 기존 관계자
톤: 간결하고 임팩트 있게. 격식체(~습니다) 사용.
길이: 카드당 2~3줄. 전체 6~8카드.
```

### 1-2. 톤 & 문체 규칙

```
✅ 올바른 톤:
- "브릿츠메디가 태국 더마 솔루션스와 독점유통 계약을 체결했습니다."
- "연 300대 규모의 계약입니다."
- "올해 하반기부터 납품이 시작됩니다."

❌ 금지:
- 반말/캐주얼체: "계약 체결함", "납품 시작!"
- 과도한 이모지 남발 (카드당 0~1개만)
- 의문형 유도: "놀랍지 않나요?" "어떻게 생각하세요?"
```

**[검수 규칙 — CRITICAL]**
카카오톡 카드뉴스의 올바른 문체는 **격식체(~습니다/~했습니다)**이다.
"~습니다"를 사용했다고 오류로 잡지 마라. 이것이 정상이다.

### 1-3. 카드 구성 (6~8카드)

```
[카드1-표지]
- 핵심 뉴스 한 줄 (15자 이내)
- 부제: 핵심 숫자 포함 (20자 이내)
- 배경: 깔끔한 단색 또는 제품 이미지

[카드2-핵심 뉴스]
- 제목: "어떤 계약인가?" 또는 팩트 요약
- 본문: WHO + WHAT + 규모 (2~3줄)
- 계약 상대방, 규모, 기간을 한 카드에

[카드3-상대방/배경]
- 파트너사 소개 (1~2줄)
- 또는 이 뉴스의 배경 (왜 이 시장인지)

[카드4-제품/기술]
- 토르RF 등 해당 제품의 핵심 특장점
- 기술 설명은 1줄로 압축
- "FDA 승인 토로이달 고주파 기술 기반 디바이스"

[카드5-의미/전망]
- 이 뉴스가 왜 중요한지 1~2줄
- 인용문이 있으면 여기에 짧게 (1문장)

[카드6-마무리]
- 향후 일정 (납품 시작 시기 등)
- 또는 "계속 주목해주세요"

[카드7-요약] (선택)
- 핵심 3줄 요약 (불릿 3개)

[카드8-CTA] (선택)
- 회사 로고 + 홈페이지/SNS 링크
```

### 1-4. 절대 포함하지 않는 것

```
- 설문조사 데이터 (공간 부족, 맥락 없이 숫자만 나열됨)
- 기술 스펙 상세 (콜라겐 수치, 온도 등)
- 경쟁사 비교
- 3줄 이상의 긴 문단
```

## 2. 이메일 뉴스레터

### 2-1. 채널 성격

```
목적: 관계자(원장님, 디스트리뷰터)에게 뉴스 + 인사이트 전달
독자: 기존 고객 원장님, 잠재 파트너사, 디스트리뷰터
톤: 전문적이되 친근한 "동료 전문가가 인사이트를 공유하는" 느낌
길이: 전체 1500~2500자
```

### 2-2. 톤 & 문체 규칙

```
✅ 올바른 톤:
- 격식체(~습니다) 기본
- "원장님, 안녕하세요. 브릿츠메디입니다."
- "이번 소식을 공유드립니다."
- "시장 진출이 어떤 의미인지 함께 살펴보겠습니다."

❌ 금지:
- 반말 ("진출했음", "체결함")
- 과도한 마케팅 톤 ("놓치지 마세요!", "지금 바로!")
- 블로그체 ("~인데요", "~거든요")
```

**[검수 규칙 — CRITICAL]**
이메일 뉴스레터의 올바른 문체는 **격식체(~습니다)**이다.
"~습니다"를 사용했다고 오류로 잡지 마라.

### 2-3. 구성

```
[제목]
- 30자 이내. 핵심 뉴스를 담되, 뉴스레터다운 톤
- 예: "태국 피부과 시장 진출, 연 300대 규모 독점유통 계약의 의미"
- 제목에 이모지 사용 금지

[프리헤더]
- 이메일 미리보기 텍스트. 40~60자.
- 제목을 보완하는 한 줄 요약
- 예: "브릿츠메디 토르RF, 더마 솔루션스와 태국 시장 파트너십 체결"

[인트로]
- 3~4문장. 인사 + 핵심 뉴스 요약 + 이 뉴스레터에서 다룰 내용 안내
- "원장님, 안녕하세요. 브릿츠메디입니다."로 시작
- 핵심 팩트(계약 상대, 규모, 시기)를 인트로에 포함

[본문1 — 뉴스 상세]
- 보도자료의 리드 + 구체 내용을 뉴스레터 톤으로 재구성
- 소제목 붙이기. 예: "연 300대 규모, 태국 독점유통 계약 체결"

[본문2 — 제품/기술 인사이트]
- 해당 제품의 기술적 의미를 원장님 관점에서 해석
- "이 기술이 원장님 시술에 어떤 의미인지" 관점
- 기술 스펙은 임상적 가치로 번역: "콜라겐 리모델링 촉진 → 시술 후 환자 만족도"
- 설문 데이터 사용 가능하나, 자연스럽게 연결될 때만

[본문3 — 시장/전략 인사이트]
- 이 뉴스의 산업적 의미
- 원장님이 알면 유용한 시장 트렌드 연결
- 예: "글로벌 시장 진출이 국내 원장님께 주는 의미"

[핵심요약]
- 불릿 3개. 각 1줄.
- 뉴스레터 전체를 요약

[CTA]
- 1개만. 명확한 행동 유도
- 예: "토르RF에 대해 더 자세히 알고 싶으시다면, 담당자에게 문의해주세요."
- 담당자: 이성호 CMO / sh.lee@britzmedi.co.kr / 010-6525-9442
```

### 2-4. 절대 포함하지 않는 것

```
- 보도자료 원문 그대로 복사
- 설문 데이터 나열 (맥락 없이 숫자만 줄줄이)
- 이미지 가이드 (뉴스레터 본문에 포함 금지)
- 이미지 생성 프롬프트 (작성자 참고용이면 별도 섹션, 본문 아닌 곳에)
```

## 3. 인스타그램

### 3-1. 채널 성격

```
목적: 브랜드 인지도 + 시각적 임팩트 + 전문성 어필
독자: 원장님, 피부과 관계자, 의료기기 관심자 (전문가 커뮤니티)
톤: 전문적이되 시각 중심. 텍스트 최소화.
형태: 캐러셀 (5~7장)
```

### 3-2. 톤 & 문체 규칙

```
✅ 올바른 톤 — 캡션:
- 격식체(~습니다) 또는 간결한 명사형 종결
- "브릿츠메디가 태국 시장에 진출합니다."
- "3년간 연 300대 규모의 독점유통 계약을 체결했습니다."

✅ 올바른 톤 — 캐러셀 슬라이드:
- 명사형 종결 또는 짧은 문장
- "태국 시장 진출 본격화"
- "연 300대 규모 독점유통"
- "FDA 승인 토로이달 고주파 기술"

❌ 금지:
- 긴 문장 (슬라이드당 3줄 이상)
- "~합니다" 남발 (슬라이드에서는 명사형이 더 적합)
- "궁금하시죠?", "놀랍지 않나요?" 같은 유도형
- 과도한 이모지 (캡션에 3개 이내, 슬라이드에 1개 이내)
```

**[검수 규칙 — CRITICAL]**
인스타그램 캡션의 "~합니다/~입니다"는 정상이다. 오류로 잡지 마라.
슬라이드 텍스트에서 "~합니다"는 info 수준으로만 "명사형 종결 권장" 안내.

### 3-3. 캐러셀 구성 (5~7장)

```
[캐러셀1-표지]
- 핵심 메시지 1줄 (10자 이내)
- 부제 1줄 (15자 이내)
- 예: "태국 시장 진출 본격화" / "토르RF 연 300대 규모 독점유통"
- 브릿츠메디 로고 포함

[캐러셀2-핵심 뉴스]
- WHAT: 무슨 일이 있었는지
- 키 넘버: "3년간 연 300대"
- 2~3줄 이내

[캐러셀3-제품/기술]
- 토르RF 핵심 특장점 1~2줄
- 시각적 아이콘/키워드 활용
- "FDA 승인" "토로이달 고주파" "콜라겐 리모델링"

[캐러셀4-의미/전망]
- 이 뉴스가 왜 중요한지 1~2줄
- "동남아 시장 확대의 첫 걸음"

[캐러셀5-CTA]
- 행동 유도: "더 알아보기 → 프로필 링크"
- 또는 "브릿츠메디 홈페이지에서 확인하세요"

[캡션]
- 2~4문장으로 뉴스 요약
- 핵심 팩트 포함
- 해시태그 15~20개 (아래 해시태그 규칙 참조)
```

### 3-4. 해시태그 규칙

```
구성: 15~20개 (23개 이상은 스팸으로 인식될 수 있음)

필수 포함:
- 자사: #브릿츠메디 #BRITZMEDI #토르RF #TORRRF
- 제품: #메디컬에스테틱 #에스테틱디바이스 #EBD

- 뉴스 관련: #태국시장 #독점유통 #글로벌진출

금지:
- 10자 이상의 긴 해시태그
- 영어+한글 혼합 해시태그 (예: #Medical에스테틱)
- 경쟁사 관련 해시태그

### 3-5. 절대 포함하지 않는 것

```
- 설문조사 데이터 (시각 채널에 숫자 나열 부적합)
- 긴 기술 설명 (슬라이드에 3줄 이상)
- 보일러플레이트 (회사 소개)
- "~세요" 명령형 CTA 남발

## 4. LinkedIn

### 4-1. 채널 성격

```
목적: B2B 전문가 네트워크에서 기업 브랜딩 + 산업 리더십
독자: 글로벌 디스트리뷰터, 의료기기 업계 전문가, 잠재 파트너사, 투자자
톤: 전문적, 데이터 기반, 산업 인사이트 중심
길이: 800~1500자 (LinkedIn 최적 길이)
언어: 한국어 기본 / 영문 병기 옵션
```

### 4-2. 톤 & 문체 규칙

```
✅ 올바른 톤:
- 격식체(~습니다) 기본
- 전문가끼리의 인사이트 공유 톤
- "이번 계약은 동남아시아 에스테틱 시장에서의 입지를 확대하는 의미가 있습니다."
- "토로이달 고주파 기술의 글로벌 경쟁력이 현지 시장에서 검증되고 있습니다."

✅ LinkedIn에서 허용되는 것:
- 산업 트렌드에 대한 전문가적 해석 (출처 명시 시)
- "이번 파트너십이 시사하는 바는..." 같은 분석적 문장
- 설문 데이터 활용 (출처 명시 필수)

❌ 금지:
- 블로그/SNS 캐주얼 톤 ("대박!", "미쳤다")
- 감정적 표현 ("정말 기쁩니다", "너무 설렙니다")
- 출처 없는 주장 ("시장에서 독보적 위치", "업계 최고 기술")
- 1인칭 감상 ("저는 이 계약이...")
```

**[검수 규칙 — CRITICAL]**
LinkedIn의 올바른 문체는 **격식체(~습니다)**이다.
"~습니다"를 사용했다고 오류로 잡지 마라.

**[검수 규칙 — 출처 검증]**
LinkedIn에서 주장하는 모든 팩트에 대해:
- 소스에 있는 팩트 → ✅
- 지식베이스에 있는 정보 → ✅ (출처 명시)
- 설문 데이터 → ✅ (출처 명시: "113명 대상 설문")
- 어디에도 없는 주장 → 🔴 critical: "출처 없는 내용"

### 4-3. 구성

```
[훅]
- 첫 2줄이 LinkedIn 피드에서 보이는 부분 (접히기 전)
- 질문형 또는 강렬한 팩트로 시작
- 예: "태국 피부과 시장, 연 300대 규모. 동남아 메디컬 에스테틱 시장이 한국 기술을 주목하는 이유는 무엇일까요?"
- 예: "3년간 연 300대 — 브릿츠메디의 동남아 시장 전략이 본격화됩니다."

[본문]
- 3~4문단. 각 문단 3~5줄.
- 1문단: 뉴스 요약 (계약 체결 사실 + 규모)
- 2문단: 배경/맥락 (왜 태국인지, 시장 전략)
- 3문단: 기술/제품 차별점 (전문가 관점)
- 4문단: 의미/전망 + 인용문 (선택)

[핵심 포인트]
- 불릿 3개. LinkedIn에서 시각적으로 잘 먹히는 구간.
- 각 1줄. 보도자료 핵심 팩트 요약.

[CTA]
- 1개. 전문가 대상.
- 예: "해외 시장 진출 시 현지 파트너와의 협업에서 가장 중요한 요소는 무엇이라고 생각하시나요? 댓글로 공유해주세요."
- 또는: "브릿츠메디 글로벌 사이트에서 더 자세한 정보를 확인하세요."

[해시태그]
- 5~8개 (LinkedIn은 해시태그 적게)
- 예: #BRITZMEDI #토르RF #MedicalAesthetics #FDA #동남아시장 #B2B #MedicalDevice
```

### 4-4. 설문 데이터 사용 규칙 (LinkedIn 전용)

```
✅ 허용:
- "브릿츠메디가 113명을 대상으로 실시한 소비자 설문에 따르면,
  재방문 1위 이유가 '효과 확실'(38.1%)이었습니다."
→ 출처 + 구체적 데이터 + 맥락 연결 OK

❌ 금지:
- "소비자들은 브랜드를 중요하게 여긴다(58.4%)" (출처 없이)
- "설문에 따르면..." 하고 보도자료와 관계없는 데이터 나열
- 설문 데이터를 3개 이상 연속 나열 (1~2개만 핵심 활용)
```

### 4-5. 절대 포함하지 않는 것

```
- 출처 없는 시장 분석 ("시장에서 독보적", "업계를 선도하는")
- 설문 데이터 3개 이상 연속 나열
- 이모지 (LinkedIn은 이모지 최소화. 0~2개)
- 보일러플레이트 전문 (간단한 회사 한 줄 소개만)
- 이미지 가이드/프롬프트 (본문에 포함 금지)
```

## 5. 네이버 블로그

### 5-1. 채널 성격

```
목적: SEO + 상세 정보 전달 + 원장님/관계자 교육
독자: 검색으로 유입되는 원장님, 디스트리뷰터, 의료기기 관계자
톤: 전문적이되 읽기 쉬운 해설 톤
길이: 2000~3500자 (네이버 블로그 최적)
```

### 5-2. 톤 & 문체 규칙

```
✅ 올바른 톤:
- 격식체(~습니다) 기본
- "이번 계약이 어떤 의미를 갖는지 살펴보겠습니다."
- "원장님들께서 주목할 포인트를 정리했습니다."
- 정보 전달 + 가벼운 해설 톤

✅ 네이버 블로그에서 허용되는 것:
- 소제목(H2) 활용 (SEO + 가독성)
- 보도자료보다 풀어서 설명
- 전문 용어에 괄호 설명 추가
- 설문 데이터 자연스럽게 인용 (출처 명시)

❌ 금지:
- "~인데요", "~거든요" (블로그라도 B2B이므로 격식 유지)
- 지나치게 캐주얼한 톤 ("꿀팁!", "대박!")
- SEO 키워드 억지 삽입 (자연스러운 흐름 내에서만)
```

**[검수 규칙 — CRITICAL]**
네이버 블로그의 올바른 문체는 **격식체(~습니다)**이다.
"~습니다"를 사용했다고 오류로 잡지 마라.

### 5-3. 구성

```
[제목]
- SEO 키워드 포함. 40자 이내.
- 예: "토르RF 태국 독점유통 계약 300대, 동남아 피부과 시장 진출 전략"
- 핵심 키워드를 앞쪽에 배치

[도입부]
- 3~5문장. 이 글에서 다룰 내용 안내.
- 독자(원장님)가 왜 이 글을 읽어야 하는지 설명
- 예: "국내 피부과 원장님들께서도 주목하셔야 할 소식을 전합니다."

[소제목1 — 뉴스 상세]
- 보도자료 핵심 내용을 블로그 톤으로 재구성
- 계약 내용, 파트너사, 규모, 일정
- 소제목 예: "3년간 연 300대 규모, 태국 독점유통 계약 체결"

[소제목2 — 기술/제품 해설]
- 원장님 관점에서 기술적 의미 풀이
- 기술 스펙을 임상적 가치로 번역
- 소제목 예: "토로이달 고주파 기술이란? 차별화된 기술의 핵심"

[소제목3 — 시장 의미/전망]
- 이 뉴스의 산업적 의미
- 원장님에게 주는 시사점
- 설문 데이터 활용 가능 (자연스럽게 연결될 때)
- 소제목 예: "동남아 시장 확대가 국내 원장님께 주는 의미"

[핵심요약]
- 불릿 3개. 블로그 전체 요약.

[CTA]
- "브릿츠메디 홈페이지에서 더 자세한 정보를 확인하세요."
- 담당자 연락처

[태그]
- 네이버 블로그 태그 8~12개
- 예: 브릿츠메디, 토르RF, 태국수출, 메디컬에스테틱디바이스, 피부과장비, ...
- 한글 태그 위주, 영문은 브랜드명만
```

### 5-4. SEO 키워드 규칙

```
필수 키워드 (자연스럽게 본문에 포함):
- 자사: 브릿츠메디, 토르RF, TORR RF
- 제품: 메디컬에스테틱, 에스테틱디바이스, EBD, 에너지기반디바이스
- 기술: 토로이달고주파, FDA승인
- 뉴스 관련: 해당 뉴스 키워드 (태국, 독점유통, 수출 등)

키워드 배치:
- 제목: 핵심 키워드 1~2개
- 도입부: 핵심 키워드 자연스럽게 1회
- 소제목: 각 소제목에 관련 키워드 포함
- 본문: 키워드 밀도 2~3% (억지 삽입 금지)
```

### 5-5. 이미지 위치 지정

```
블로그 본문에 이미지를 넣을 위치를 [IMAGE: 설명] 형식으로 표시.
이미지 가이드는 본문 밖 별도 섹션에.

예:
[IMAGE: 브릿츠메디-더마 솔루션스 계약 체결 현장 사진]

위치 규칙:
- 도입부 직후 1장 (대표 이미지)
- 소제목2 (기술 해설) 앞뒤 1장 (제품 이미지)
- 본문 중 2~3장이 적절
```

### 5-6. 절대 포함하지 않는 것

```
- 보도자료 원문 그대로 복사
- 설문 데이터 3개 이상 연속 나열
- 이미지 생성 프롬프트 (본문에 포함 금지)
- 보도자료의 [회사 소개] 전문 복사 (한 줄 소개만)
- "행사 기간" 등 보도자료 용어 (블로그 톤으로 전환)
```

## 7. 전 채널 검수 규칙 통합

### 7-1. 문체 검수 — 절대 오류로 잡지 않는 것

```
[CRITICAL — 검수 시스템 필독]

아래는 각 채널에서 정상적인 표현이다.
이것들을 "보도문체 위반"으로 잡으면 안 된다:

✅ 모든 채널: "~습니다", "~했습니다", "~입니다" → 정상 (격식체)
✅ 카카오톡: "~습니다" → 정상
✅ 뉴스레터: "원장님, 안녕하세요" → 정상 (독자 호칭)
✅ 인스타그램 캡션: "~합니다" → 정상
✅ LinkedIn: "~습니다" → 정상
✅ 블로그: "~습니다", "살펴보겠습니다" → 정상

오류로 잡아야 하는 문체:
🔴 보도자료에서 "~습니다" 사용 → 보도문체(~했다) 위반
🔴 모든 채널에서 반말 사용 → "계약 체결함", "납품 시작!"
🔴 모든 채널에서 "~인데요", "~거든요" → B2B에 부적합
```

### 7-2. 팩트 검수

```
[전 채널 공통]
- 보도자료의 핵심 팩트(숫자, 날짜, 이름)가 정확히 전달되는지
- 보도자료에 없는 팩트가 추가되지 않았는지
- 의료법 금지어가 사용되지 않았는지
- 출처 없는 주장이 없는지

[채널별 추가]
- LinkedIn: 모든 주장에 소스 근거 확인
- 블로그: SEO 키워드가 자연스럽게 포함되는지
- 인스타그램: 해시태그 개수 15~20개 범위인지
```

### 7-3. 구조 검수

```
각 채널의 필수 섹션이 빠지지 않았는지 확인:

카카오톡: 표지 + 핵심뉴스 + CTA (최소 3카드)
뉴스레터: 인트로 + 본문1 + 핵심요약 + CTA
인스타그램: 표지 + 2~3슬라이드 + CTA + 캡션 + 해시태그
LinkedIn: 훅 + 본문 + 핵심포인트 + CTA
블로그: 도입부 + 소제목 2~3개 + 핵심요약 + CTA + 태그
```

## 8. 이미지 가이드 / 이미지 프롬프트 처리 규칙

### 8-1. 원칙

```
이미지 가이드 = 작성자가 어떤 사진을 준비해야 하는지 안내
이미지 프롬프트 = AI 이미지 생성용 영문 프롬프트

둘 다 "작성자 참고용"이다. 
최종 콘텐츠 아웃풋(복사, 내보내기)에는 포함하지 않는다.
```

### 8-2. 채널별 처리

```
카카오톡: 이미지 가이드 → 별도 섹션 (📌 작성자 참고용)
뉴스레터: 이미지 가이드 → 별도 섹션
인스타그램: 이미지 프롬프트 → 별도 섹션 (캐러셀 디자인 참고)
LinkedIn: 이미지 가이드 → 별도 섹션
블로그: [IMAGE: 설명]은 본문에 위치 표시 / 상세 가이드는 별도 섹션

"전체 복사" 기능 사용 시:
→ 이미지 가이드 섹션 제외하고 복사
→ 이미지 프롬프트 섹션 제외하고 복사
→ 순수 콘텐츠만 복사
```

## 9. 카테고리별 채널 콘텐츠 포커스

보도자료 카테고리에 따라 각 채널이 강조할 포인트가 다르다:

### 전시회·학회

```
카카오톡: "어디서, 무엇을 전시하는지" + 부스 번호
뉴스레터: 전시 제품의 임상적 의미 + 원장님 관전 포인트
인스타그램: 전시 현장감 (부스, 제품 사진) + "현장에서 만나요"
LinkedIn: 학회의 산업적 위상 + 기술 발표 내용
블로그: 전시 제품 상세 해설 + 학회 의미 + 향후 일정
```

### 파트너십·계약

```
카카오톡: "누구와, 어떤 규모로" 핵심만
뉴스레터: 파트너십이 원장님에게 주는 의미 (글로벌 검증)
인스타그램: 계약 현장 + 핵심 숫자 비주얼
LinkedIn: 시장 전략 분석 + 파트너사 역량 + 산업 인사이트
블로그: 파트너사 상세 + 시장 배경 + 향후 계획
```

### 신제품·신기술
