## 설계 원칙

1. **마케터 입력은 카톡 수준** — 대충 적어도, 클릭만 해도, 사진만 올려도 콘텐츠가 나온다.
2. **각 성격마다 페르소나** — 같은 AI가 쓰더라도 작성자의 캐릭터가 느껴져야 한다.
3. **모바일 우선** — 휴대폰에서 콘텐츠 제작→배포까지 완결.
4. **비용 투명** — 모든 API 호출의 토큰/원가를 실시간 표시.
5. **britzmedi.com 통합** — Content Factory를 britzmedi.com/admin에 포함, 홈페이지도 배포처.

### 단문 채널의 CTA 전략

Instagram, 카카오톡처럼 장문이 어울리지 않는 채널:
```
[핵심 메시지 2~3문장 요약]
...
👉 자세한 내용은 여기서 확인하세요
🔗 www.britzmedi.co.kr/[콘텐츠별 URL]

- 핵심 써머리만 제공하고 CTA로 britzmedi.co.kr의 해당 콘텐츠로 유도
- 홈페이지에 풀 콘텐츠가 있어야 CTA가 의미 있음
- → **홈페이지 채널은 대부분의 콘텐츠에서 자동 선택되어야 하는 이유**

## 1. 📑 논문/연구 해설

### 목적
최신 피부과/에스테틱 논문을 BRITZMEDI 관점에서 해설하여
원장님·업계 전문가에게 교육적 가치를 제공하는 콘텐츠.

### 출처 (Sources)

| 출처 | 용도 | API/방법 |
|------|------|---------|
| **PubMed** | 피어리뷰 논문 검색 (1차 소스) | E-utilities API (무료) |
| **bioRxiv** | 프리프린트 최신 연구 | bioRxiv MCP (연결됨) |
| **Google Scholar** | 보완 검색 | 웹 검색 |
| **논문 원문 PDF** | 초록+본문+장표 추출 | PDF 파싱 |
| **BRITZMEDI 지식베이스** | 제품-논문 연결 | 내부 DB |

### AI 역할 분담

| 단계 | AI | 역할 |
|------|-----|------|
| 논문 검색 | PubMed API + bioRxiv MCP | 키워드 기반 자동 검색 |
| 논문 수집 | 웹 크롤링 | PMC 전문, PDF 다운로드 |
| 논문 분석 | Claude Sonnet 4.5 | 초록/본문 읽고 요약 + 제품 연결 + 영향력 평가 |
| 장표 추출 | Claude Vision (이미지 분석) | 논문 내 Figure/Table 추출 + 한글 설명 |
| 콘텐츠 생성 | Claude Sonnet 4.5 | 채널별 교육 콘텐츠 작성 |
| 검수 | Claude Sonnet 4.5 | 팩트 날조 여부 + 의료법 위반 검수 |

### 논문 내 장표/이미지 추출

```
논문 PDF 업로드 또는 PMC 풀텍스트 접근 시:

1. PDF에서 Figure/Table 위치 자동 감지
2. Claude Vision으로 각 Figure/Table 분석:
   - Figure 1: 콜라겐 밀도 변화 그래프
     → "시술 전후 콜라겐 밀도가 23% 증가한 것을 보여주는 막대그래프"
   - Table 2: 환자군별 개선 점수
     → "32명 환자의 탄력 개선 점수 비교표"
3. 콘텐츠에 적합한 장표를 자동 선택
4. 한글 캡션 자동 생성
5. 채널별 활용:
   - 블로그/홈페이지: 원본 Figure + 한글 캡션
   - LinkedIn: 핵심 Figure 1개 + 인사이트
   - 뉴스레터: Figure 요약 텍스트로 대체
   - 인스타: Figure를 카드뉴스 소재로 재구성
```

## 2. 🏥 납품/도입 사례

### 자동화 전략

```
마케터 입력: 병원명 + 제품 선택 (2개만)
+ 선택: 메모 한 줄 ("추가 구매", "3대째")

AI가 자동으로:
1. 제품 DB에서 제품 스펙/특장점 로드
2. 10가지 패턴 코멘트 중 상황에 맞는 것 자동 채택
3. 채널별 납품 축하 콘텐츠 생성
```

3. 원장님 인터뷰 직후 녹음 파일 → 바로 업로드 → 콘텐츠 생성

// Web Share API — 모바일에서 "공유하기" 시트 열기
if (navigator.share) {
  await navigator.share({
    title: '새 사무실 이전 소식',
    text: generatedContent,
    // url: 'https://britzmedi.co.kr/...'  // 홈페이지 링크 포함 가능
  });
}
```
