## 1. PDF 보도자료 — 이미지 누락 문제

### 잘된 점 ✅

- **라벨 문제 완전 해결:** "제목/부제목/본문" 라벨이 사라지고, "보 도 자 료 / PRESS RELEASE" 헤더가 정상 출력
- **구조 완성도 높음:** 제목 → 부제목 → 본문 → 대표 인용문 → 회사 소개 → 회사 정보 테이블 → 미디어 문의 순서가 전문 보도자료 양식에 정확히 부합
- **회사 정보 테이블:** 회사명/대표이사/본사/홈페이지가 깔끔하게 정리됨
- **연락처 정확:** 이성호 CMO / sh.lee@britzmedi.co.kr / 010-6525-9442 ✅

### 문제점 ❌

**이미지 누락이 유일하지만 치명적인 문제입니다.**

Word 파일에는 이미지가 정상 포함되어 있음을 확인했습니다 (word/media/ 폴더에 77KB PNG 파일 존재).
그러나 PDF(브라우저 인쇄)에서는 이미지가 출력되지 않습니다.

**원인 추정:**
`openPrintView` 함수가 HTML을 생성할 때 이미지 태그를 포함하지 않거나,
Supabase Storage URL을 사용하는 이미지가 인쇄 시점에 로드되지 않는 것으로 보입니다.

### Claude Code 수정 지침

```
파일: src/components/create/Create.jsx → openPrintView 함수

확인 사항:
1. openPrintView에서 이미지 URL(Supabase Storage)을 <img> 태그로 포함하는지 확인
2. 이미지가 있을 경우 인쇄 HTML에 포함되도록 수정
3. 이미지 로딩 완료 후 print() 호출되도록 onload 핸들러 추가
4. Word 다운로드(generatePressReleaseDocx.js)에는 이미지가 포함되므로
   동일한 이미지 URL을 openPrintView에서도 사용

수정 방향:
- v2Content에서 이미지 URL 추출
- openPrintView HTML에 <img src="이미지URL" style="max-width:100%; margin:20px 0"> 삽입
- window.onload 또는 img.onload 후 window.print() 호출
```

## 6. 링크드인 — 80/100 (가장 큰 개선)

### 잘된 점 ✅

**① 한영 혼용이 자연스럽고 전문적**

"이번 exclusive distribution 계약이 의미 있는 이유는, 단순히 제품을 수출하는 것이 아니라 현지에서 검증된 파트너와 함께 체계적인 go-to-market strategy를 실행한다는 점입니다."

이 문장은 링크드인에서 볼 수 있는 전형적인 비즈니스 포스트 톤입니다. "exclusive distribution"과 "go-to-market strategy"가 한국어 문맥에 자연스럽게 녹아 있어요. V1에서는 영어가 전혀 없었던 것과 비교하면 큰 변화입니다.

**② 비즈니스 인사이트 섹션이 실질적**

V1: (인사이트 없음, 보도자료 팩트 나열)
V2: "동남아시아 메디컬 에스테틱 시장은 중산층 확대와 미용 의료에 대한 관심 증가로 빠르게 성장하고 있습니다. 하지만 시장 진입에는 현지 규제 이해, 유통망 구축, 사후관리 체계 등 여러 장벽이 존재합니다."

이것이 링크드인 콘텐츠의 핵심입니다. "성장 중이다"라는 피상적 진술이 아니라, "장벽도 있다"는 현실적 분석을 제공합니다. 읽는 사람이 "이 사람은 현장을 안다"고 느끼게 만드는 문단입니다.

**③ Engagement Question으로 마무리**

"여러분의 회사는 해외 시장 진출 시 현지 파트너 선정에서 어떤 기준을 가장 중요하게 보시나요?"

링크드인 알고리즘에서 댓글 유도는 도달률에 직접 영향을 줍니다. V1에는 없었던 이 질문이 포스트의 완성도를 크게 높입니다.

**④ 해시태그 전략이 정확**

#BRITZMEDI #TORRRF #MedicalAesthetics #MedicalDevice #Dermatology #EBD #Thailand #MarketExpansion #Partnership #B2B #의료기기 #메디컬에스테틱
→ 영문 10개 + 한글 2개 = 12개. 글로벌 오디언스 타깃에 적합

**⑤ 글자수 적정: 915자 (목표 800~1,200자)**

### 보강 필요 사항

**① 훅 라인에서 이모지 렌더링 문제**

현재: "BRITZMEDI x Derma Solutions — 3년간 연 300대 규모, 태국 피부과 시장 본격 진출 TH"
→ "TH"는 🇹🇭 (태국 국기 이모지)가 텍스트로 변환된 것

```
프롬프트 보강:
"국기 이모지를 사용할 때는 유니코드 이모지(🇹🇭)를 직접 사용하세요.
텍스트 코드(TH, US 등)로 대체하지 마세요."

코드 레벨 확인:
stripMarkdown 함수 또는 channelGenerate.js에서 이모지를
텍스트로 변환하는 로직이 있는지 확인.
이모지가 손실되는 경로를 추적하여 수정.
```

**② "미용 의료" 표현 수정**

현재: "미용 의료에 대한 관심 증가로"
→ "미용"은 금지 용어 계열. "메디컬 에스테틱 시술에 대한 수요 증가로"가 적합

```
프롬프트 보강:
금지 용어 목록에 "미용 의료"를 추가:
"미용 의료" → "메디컬 에스테틱 시술" 또는 "메디컬 에스테틱"
```

**③ 인사이트를 "우리가 배운 것" 관점으로 한 단계 더**

현재 인사이트는 "시장 분석" 수준입니다. 링크드인에서 가장 공유/좋아요가 많은 포스트는 "우리는 이 과정에서 이런 걸 배웠다"는 개인/회사의 학습 포인트입니다.

현재: "시장 진입에는 현지 규제 이해, 유통망 구축, 사후관리 체계 등 여러 장벽이 존재합니다."
→ 좋지만, 일반론

개선: "3년간의 파트너 탐색 끝에 배운 것은, 현지 시장을 아는 파트너가 곧 시장 진입 속도를 결정한다는 점이었습니다."
→ 구체적 경험 기반 인사이트

```
프롬프트 보강:
"인사이트 섹션에서는 일반적인 시장 분석이 아닌,
'이 과정에서 우리가/업계가 배운 교훈'을 1~2문장 포함하세요.
예: '이번 파트너십을 통해 확인한 것은, 현지 유통 전문성이
제품 기술력만큼 중요하다는 점이었습니다.'
```
